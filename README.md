# Sistema de Controle de Estoque – Floricultura

Sistema web desenvolvido para o controle de estoque de uma floricultura, permitindo o gerenciamento simples e organizado dos produtos disponíveis. O projeto possibilita o cadastro, a listagem e a remoção de itens por meio de uma interface web.

## Funcionalidades

- Cadastro de produtos no estoque  
- Listagem dos produtos cadastrados  
- Remoção de produtos  
- Interface web utilizando HTML e CSS   

## Tecnologias Utilizadas

- Python  
- Flask  
- HTML  
- CSS  

## Como Executar o Projeto

1. Ative o ambiente virtual: venv\Scripts\activate
2. Instale as dependências: pip install flask 
3. Execute a aplicação: python app.py
